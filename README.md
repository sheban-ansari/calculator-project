# calculator-project

# Stylish Calculator Web App

A modern, responsive and stylish calculator built using **HTML, CSS and JavaScript**.  
This project performs basic arithmetic operations with a clean glassmorphism UI.

---

## Features

-  Addition  
-  Subtraction  
-  Multiplication  
-  Division  
-  Percentage calculation  
-  Clear All (AC)  
-  Delete single digit (DEL)  
-  Fast & smooth UI  
-  Fully responsive design
